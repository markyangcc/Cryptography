# Stream Cipher with RC4
Implementation of RC4 encryption and decryption in C++

## How to run

### 1、build it
```bash
g++ main.cpp -o main
```

### 2、run it
```bash
./main <anyfile name>
(For example, ./main data.txt)
```
Then choose options of encrypt or decrypt, and hit enter.
Follow the instruction printed on the screen.
